# SQL Data Warehouse & Analytics Project

## Overview
This project demonstrates the design and implementation of a modern Data Warehouse using SQL Server.

## Project Objectives

### Data Engineering
- Import ERP and CRM data from CSV files
- Clean and validate data
- Integrate data into a unified model

### Data Analytics
- Customer Behavior Analysis
- Product Performance Analysis
- Sales Trend Analysis

## Architecture
### Bronze Layer
Raw source data

### Silver Layer
Cleaned and transformed data

### Gold Layer
Business-ready fact and dimension tables

## Tech Stack
- SQL Server
- T-SQL
- ETL
- Data Warehousing

## Skills Demonstrated
- SQL Development
- Data Engineering
- Data Modeling
- Data Analytics
